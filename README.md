#  Places Sharing and photos  Back-end

A simple RESTful API for a platform where user can share places, and update photos and describe locations and locations on the website

## Tech Stack
- **Node.js**
- **Express** 4
- **MongoDB** with Mongoose
- **body-parser**

 ## Requisites
- Node.js installed
- MongoDB instance (local or Atlas)

##  Getting Started
1. Clone the repository
   git clone https://github.com/Murilets/MongoDB-connection.git

2. Install dependencies
   
    **npm install**

3. Create a .env file in the root with your MongoDB connection string

   **MONGO_URI=your_mongodb_connection_string**

4. Start the devlopment server

   **npm start**
